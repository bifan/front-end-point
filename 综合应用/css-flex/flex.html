CSS flex 示例

flex 对齐姿势😄:

justify-content  →  Main 轴对齐
align-items      →  Cross 轴对齐(只有一行的情况)
align-content    →  Cross 轴对齐(多行时)

Cross 轴有多行且有多余空间时可使用align-content 对齐

有多余空间就的默认情况(行之间不会紧贴着):

align-content: start

align-content: center

align-content: end

ac: space-evenly

ac: space-around

ac: space-between

flex item 跨行 ❌

换行后并不会补空隙:

这就是flex 的性质, 可以在行内动来动去, 但是无法跨行
改变方向或order 都解决不了item 大小不一出现的空隙
跨行请使用Grid
另, 实现Masonry 布局只用Grid 或flex 都不行, 还是得靠JS 配合:
Approaches for a CSS Masonry Layout
